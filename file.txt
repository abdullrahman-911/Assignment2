Q1:fork() creates a new process (child) by duplicating the existing process (parent).
In the parent process, fork() returns the PID (Process ID) of the child.

In the child process, fork() returns 0.

If fork() fails, it returns -1.

Q2: the job of the Linker is :
The linker is a crucial tool in the software development process that combines various compiled object files into a single executable program. Its main job is to ensure that all the separate pieces of code and data in a program are correctly connected and able to work together

Q3: the job of loader:

Allocate memory for the code, data, stack, and heap.

Relocate addresses to adjust relative references to actual memory locations.

Link dynamic libraries and resolve references to functions/variables in them.
